When a Fixture Makes Sense

  • The job runs repeatedly
  • Setup time is significant
  • Part location varies
  • Multiple parts could be loaded at once
  • The machine is capacity constrained
  • Skilled operators are required to recreate the setup

Process

1. Review

Review the part, current workholding, machine, run frequency, and setup pain.

2. Design

Develop practical fixture concepts around actual operator and machine constraints.

3. Build Package

Deliver CAD, drawings, BOM, hardware references, and setup notes.

4. Support

Support quoting, build questions, revisions, and validation planning.
